# Technical Assignment

## Техническое задание для разработки программы "Калькулятор"

**Цель:** реализовать программу с консольным интерфейсом.

**Заказчик:** -

**Разработчик:** Lobotomy Corp.

Реализовать функционал выполнения арифметических операций:

* сложения двух чисел;
* вычитания двух чисел;
* умножения двух чисел;
* деления двух чисел;
* возведения целого числа в степень.


## Механизм взаимодействия пользователя с программой:

1. Программа отображает меню выбора операций или выхода из программы.

2. Пользователь выбирает операцию или выход из программы.

3. Программа запрашивает, а пользователь осуществляет ввод требуемых для
выполнения операции.

5. Программа выполняет вычисления, и отображает результат вычисления.

6. Программа заново отображает меню выбора операций или выхода из программы.


## Требования к реализации

* Программа должна быть реализована на языке C#.

* Программа должна поддерживать запуск и работу на нескольких ОС, включая
Windows и Linux.

* Программа должна осуществлять проверку пользовательского ввода на
корректность.

* Программа должна поддерживать ввод и вычисление дробных (double) чисел в
диапазоне от -2^(64-1) до 2^(64-1), если не указано иное.
